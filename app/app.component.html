Dev Force 2.0 FE Challenge

Hi there!

This is one is going to take a little bit longer than the usual CodeWars, but I hope you find value in the challenge. Don't be afraid to ask me questions if you need a bit of assitance!

I have shared a .zip file with you on the slack channel. The folder stucture is as follows:

  • README.md (READ THIS FIRST)
  • design (here you will find previews of the mobile and desktop designs)
  • assets (all of the necessary assets required to mock the designs)
  • README-template.md
  • index.html
  • style-guide.md

Make use of Bootstrap within your design.

Make sure your design is mobile friendly. You can use Chrome dev tools' built in screen emulator to test the mobile view (its just to the left of the Chrome dev tools console tab).

You can fork this StackBlitz in the top left corner and start your design!
